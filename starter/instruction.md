# Copilot Instructions

You are assisting with a Flask Sudoku refactor project.

Project rules:
- Keep changes small and testable.
- Preserve existing working behavior unless replacing it deliberately.
- Use clear modular Python functions.
- Use readable JavaScript for browser-side game features.
- Do not add unnecessary frameworks.
- Sudoku puzzles must have exactly one unique solution.
- Difficulty levels must control how many cells are prefilled.
- Prefilled cells must be locked.
- Invalid moves should give immediate visual feedback.
- Completed games should show a success message.
- Top 10 scores should be saved in localStorage.
- UI must support responsive layout, light mode, and dark mode.
- Keep code beginner-readable and add comments where useful.
